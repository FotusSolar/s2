## VOUCHER GERAL [[link](https://sandbox.solaryum.com.br/fotus-yfe/configuracoes/vouchers)]

CRUD no Solaryum para criação de vouchers de promoção, contendo os campos:

- Código
- Descrição
- Prazo de validade (dias)
- Valor mínimo do pedido
- Tipo de desconto
    - % sobre o pedido
    - R$ sobre o pedido
- Valor do desconto
- Benefícios
- Condições
