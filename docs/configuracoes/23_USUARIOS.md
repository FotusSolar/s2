#### USUÁRIOS

CRUD dos usuários do sistema no Solaryum, que podem ser:
- gerentes
- supervisores de time
- vendedores
- administrativo
- financeiro

O formulário contém os seguintes campos:

- Nome
- E-mail
- Telefone
- Senha
- [ ] Ativo
- Time
    - [ lista de times com hierarquia ]
- Centro de distribuição
    - [ lista de centros de distribuição ]
- Papel
    - Administrador
    - Gerente
    - Supervisor
    - Vendedor
    - Financeiro
    - Logística
    - Suporte
- Permissões
    - [ ] Ver loja
    - [ ] Adicionar cliente
    - [ ] Gerar cotações
    - [ ] Aprovar descontos
    - [ ] Configuração

    - [ ] Ver pedidos de outros times
